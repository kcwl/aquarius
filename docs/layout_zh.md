# 关键字类型
## 命名空间
命名空间包括 namespace。命名空间用来区分不同类型，避免命名冲突。当前文件中有且只有一个命名空间，不支持多级命名空间。

使用方法：
```
namespace animal;   //声明一个命名空间，名字是animal
```

# 变体类型
变体类型是根据不同的数据的实际值，移除字节前的`0`,达到节省空间的目的，在1个字节中，符号位用来表示是否有数据，其他7位表示实际值。


# 基本类型
基本类型包括： 数值类型，布尔类型和字符串类型。

## 数值类型
数值类型包括：有符号类型，无符号类型，浮点类型。有符号类型和无符号类型用来表示整数，浮点类型用来表示小数。数值类型均采用变体类型进行编码。在序列化时，会根据实际值的大小，选择合适的位进行数据存储。

### 有符合类型
有符号类型包括：int32，int64。
+ 相同点
  + int32和int64都使用zigzag算法进行数据编码，以减少数据存储空间。
+ 不同点
  + int64 默认会直接填充前4个字节，标记前4字节已经存在数据，如实际值不足4个字节，则退化为int32来存储实际数值。

使用方法：
```
int32 age;  // 声明一个int32类型的变量，名字是age
int64 id;   // 声明一个int64类型的变量，名字是id
```

### 无符号类型
无符号类型包括：uint32，uint64。
+ 不同点
  + uint64 默认会直接填充前4个字节，标记前4字节已经存在数据，如实际值不足4个字节，则退化为uint32来存储实际数值。

使用方法：
```
uint32 address; // 声明一个uint32类型的变量，名字是address
uint64 id;      // 声明一个uint64类型的变量，名字是id
```

### 浮点类型
浮点类型包括：float，double。浮点类型均按照IEEE-754标准进行存储。

使用方法：
```
float weight;   // 声明一个float类型的变量，名字是weight
double height;  // 声明一个double类型的变量，名字是height
```

## 布尔类型
布尔类型包括：bool。每一个bool类型占用1个字节，0表示false，非0值表示true。布尔类型在序列化时，会直接填充1个字节。

使用方法：
```
bool is_alive; // 声明一个bool类型的变量，名字是is_alive
```

## 字符串类型
字符串类型包括：string。字符串采用变长数组的方式，由字符串长度和字符串内容两部分组成，在序列化时，每个字符占1个字节。

使用方法：
```
string name; // 声明一个string类型的变量，名字是name
```

# 复合类型
复合类型包括：字节序列，变长数组，枚举类型，消息类型(请求类型和响应类型)

## 字节序列
字节序列包括：bytes。字节序列采用变长数组的方式，由字节长度和字节内容两部分组成，在序列化时，每个字节占1个字节。

使用方法：
```
bytes data; // 声明一个bytes类型的变量，名字是data
```

## 变长数组
变长数组类型包括：repeated。变长数组采用len-value的方式进行编码，在序列化时，每个元素的编码方式取决于元素类型。

使用方法：
```
repeated int32 age; // 声明一个int32类型的变长数组，名字是age
```

## 枚举类型
枚举类型包括：enum。枚举类型和int32类型采用相同的编码方式，在序列化时，会根据实际值的大小进行数据存储。

使用方法：
```
enum color 
{
    red;
    green;
    black;
};
```

# 结构类型
结构类型是序列化中的基本单位，包括：结构体类型， 消息类型(请求消息类型和响应消息类型)。

## 结构体类型
结构体类型使用message表达，结构体内部可以有多个字段，每个字段都有自己的类型和名称。结构体类型在序列化时，会根据字段的类型进行数据编码。

使用方法：
```
message person
{
    int32 age;
    string name;
};

```

## 消息类型
消息类型包括：请求消息类型和响应消息类型。用来表示客户端和服务器之间的通信协议体。

使用方法：
```
rpc(moudle) use_login = 10001     //声明一个rpc消息，名字是use_login，协议体唯一标识为10001
{
    request                 //声明一个请求消息类型
    {
        uint32 id;
    };

    response                //声明一个响应消息类型
    {
        person info;
    };
}

```
